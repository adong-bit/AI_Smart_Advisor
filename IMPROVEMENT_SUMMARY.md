## 🔴 发现的5个严重问题

### 1️⃣ **词汇库极度不足** 
- 仅检查 **4个词**：涨、利好、增长、突破
- 所有其他情况一律判为利空
- 导致大量误判

**实例**:
```
"央行宣布降准" → 缺少"降准"词汇 → 判为利空 ❌（应该是利好）
"市场担忧" → 缺少"担忧"词汇 → 判为利空 ❌（应该中性偏空）
```

### 2️⃣ **无法处理否定词** 🚨 **最严重**
- 完全无法识别否定前缀
- "未能"、"没有"、"不会"等词无效

**致命实例**:
```
"未能突破关键阻力位"
→ 包含"突破" → 判为 POSITIVE ❌
→ 实际应该是 NEGATIVE ✓
→ 这个错误非常严重！

"没有增长"
→ 包含"增长" → 判为 POSITIVE ❌
→ 实际应该是 NEGATIVE ✓
```

### 3️⃣ **利空词库完全缺失**
缺少的利空词导致无法捕捉负面信息：
```
下跌、暴跌、跌停、大幅下跌
风险、警告、担忧、困难
停产、破产、违约、裁员
收紧、加息、延迟
```

### 4️⃣ **多空信息无法平衡**
```
"央行释放流动性但市场担忧后续政策收紧"
→ 既有利好、又有利空
→ 简单判法无法平衡 → 错误的二值判断
```

### 5️⃣ **无置信度量化**
```
"政策利好" → POSITIVE
"美联储暗示降息" → POSITIVE

两者都返回POSITIVE，但实际确定性差异巨大！
```

## ✅ 完整改进方案

### 改进的 `analyze_news_sentiment()` 函数

#### 1. 扩充词汇库
```
【强利好词】(权重3分)
涨停、暴涨、大幅上升、创新高、利好、回购、增持、利润增长

【温和利好词】(权重1分)
上涨、反弹、修复、改善、增长、回暖、扩张、提振、稳定

【强利空词】(权重3分)
跌停、暴跌、大幅下跌、创新低、风险、警告、停产、破产、裁员

【温和利空词】(权重1分)
下跌、回落、走弱、承压、衰退、下滑、萎缩、削减

【宏观词汇】
降息→利好、加息→利空、降准→利好、收紧→利空
```

#### 2. 否定词检测与反转
```python
negation_words = ["未", "没有", "不会", "不能", "缺乏", "无法"]

if any(neg in title for neg in negation_words):
    if 检测到利好词:
        反转为利空
    if 检测到利空词:
        反转为利好
```

#### 3. 加权评分
```python
bullish_score = 强利好*3 + 温和利好*1
bearish_score = 强利空*3 + 温和利空*1

if bullish_score > bearish_score:
    sentiment = "positive"
elif bearish_score > bullish_score:
    sentiment = "negative"
else:
    sentiment = "neutral"  # 新增中性判断
```

#### 4. 置信度计算
```python
confidence = min(0.95, 0.5 + score * 0.15)
# 返回 0.0~1.0 的置信度
# 0.9+ = 高置信度
# 0.6~0.8 = 中置信度  
# 0.5~ = 低置信度
```

#### 5. 返回值结构
```python
return {
    "sentiment": "positive|negative|neutral",
    "confidence": 0.0-1.0,      # 置信度
    "reason": "判断理由",        # 可解释
    "has_negation": bool,        # 是否检测到否定词
    "bullish_score": int,        # 利好得分
    "bearish_score": int,        # 利空得分
}
```

## ⚠️ 已知局限（后续优化）

虽然改进很大，但仍有局限：

1. **行业特异性** - 同样新闻对不同板块影响不同
   - 例：油价上升 → 石油企业利好，航空企业利空
   
2. **上下文联系** - 目前只看标题，不读正文
   - 例：复杂的因果关系无法完全捕捉

3. **时间效应** - 无法区分预期vs实际
   - 例：市场早已消化的消息影响不同

4. **NLP深度** - 没有使用神经网络
   - 后续可考虑集成BERT等深度学习模型
