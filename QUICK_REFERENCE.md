## 核心问题和解决方案一页纸速查表

### ❌ 原问题（改进前）
```
代码位置: app.py 第1237行、1263行

原判断方式:
"sentiment": "positive" if any(x in title for x in ["涨", "利好", "增长", "突破"]) else "negative"

问题:
1. 词汇库仅4个词 → 大量漏判
2. 无否定词处理 → "未能突破"误判为利好 ❌
3. 无利空词库 → 所有非利好词都判为利空
4. 无多空平衡 → 好坏词等量时仍判为利好或利空
5. 无置信度 → 无法表达判断确定性
```

### ✅ 新判断方式（改进后）
```
函数: analyze_news_sentiment(title)

返回结构:
{
    "sentiment": "positive|negative|neutral",
    "confidence": 0.0-1.0,
    "reason": "判断理由",
    "bullish_score": int,
    "bearish_score": int,
    "has_negation": bool
}

流程：检查否定词 → 词汇计分 → 反转处理 → 判决 → 置信度
```

## 🔑 关键词快速查表

### 利好词

**强(3分)**: 涨停、暴涨、突破、利好、回购、增持  
**弱(1分)**: 上涨、反弹、增长、改善、回暖、稳定

### 利空词

**强(3分)**: 跌停、暴跌、风险、警告、停产、破产、裁员  
**弱(1分)**: 下跌、承压、衰退、下滑、困难、延迟

### 宏观词

**利好**: 降息、降准、释放流动性  
**利空**: 加息、收紧、缩表

### 否定词

未、没有、不会、不能、缺乏、无法
