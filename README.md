# 智投星 - AI 智能投顾平台 Demo

> 一站式 AI 驱动的智能投顾解决方案，展示 AI 技术在财富管理领域的核心应用场景。

## 产品定位

面向证券公司/财富管理机构的智能投顾平台，利用机器学习、自然语言处理等 AI 技术，为投资顾问和终端客户提供个性化的资产配置、智能选股、投资咨询等服务。

## 核心功能模块

| 模块 | 功能 | AI 技术 |
|------|------|---------|
| 市场总览 | 实时行情、板块热力图、AI 市场洞察 | NLP 情绪分析、时序预测 |
| 风险测评 | KYC 问卷、风险画像、合规适当性管理 | 分类模型、用户画像 |
| 智能配置 | 个性化资产配置方案、历史回测 | 现代投资组合理论(MPT)、优化算法 |
| 智能选股 | 多因子选股模型、因子权重可调 | 多因子模型、特征工程 |
| 组合管理 | 持仓分析、风险监控、AI 调仓建议 | 风险模型、异常检测 |
| AI 助手 | 投资咨询对话、意图识别 | 大语言模型、意图分类、NLP |
| 投教中心 | 投资知识体系、AI 技术科普 | 知识图谱、推荐系统 |

## 快速启动

```bash
# 1) 进入项目目录
cd ~/Desktop/智能投顾助手

# 2) 创建虚拟环境（仅首次需要）
python3 -m venv .venv

# 3) 激活虚拟环境
source .venv/bin/activate

# 4) 安装依赖
python -m pip install -r requirements.txt

# 5) 启动应用
python app.py

# 6) 打开浏览器访问
# http://127.0.0.1:5008
```

> 退出虚拟环境可执行：`deactivate`

## 技术架构

- **后端**: Python Flask
- **前端**: HTML5 + CSS3 + JavaScript
- **图表**: ECharts 5.x
- **设计**: 深色金融专业主题

## 产品设计亮点

1. **合规先行**: 投资前必须完成风险测评，符合监管适当性管理要求
2. **个性化推荐**: 基于用户风险画像定制资产配置方案
3. **可解释 AI**: 选股结果附带 AI 理由说明，增强用户信任
4. **专业指标**: 夏普比率、最大回撤、VaR 等专业风控指标
5. **全链路覆盖**: 从测评 → 配置 → 选股 → 管理 → 咨询的完整投资链路

## 面试展示建议

1. 从「市场总览」开始，展示平台整体设计
2. 进入「风险测评」，演示 KYC 合规流程
3. 查看「智能配置」，讲解 MPT 资产配置理论
4. 操作「智能选股」，调节因子权重展示多因子模型
5. 打开「AI 助手」，演示 NLP 驱动的投顾对话
6. 最后到「投教中心」，展示 AI 技术栈理解
